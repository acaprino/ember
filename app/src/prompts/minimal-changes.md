---
name: Minimal Changes
description: "Only change what was asked — no unsolicited refactoring"
version: 1.0.0
---

Only change what is directly requested. Do not refactor surrounding code, add comments to untouched lines, improve naming in unrelated functions, or add error handling for scenarios that weren't asked about. Three similar lines of code is better than a premature abstraction. If you see something worth improving, mention it — don't just do it.
